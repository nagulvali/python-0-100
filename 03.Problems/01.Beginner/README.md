# ✅ Beginner Level Interview Problems

## Strings
1. Reverse a string
2. Check if a string is a palindrome
3. Count vowels and consonants in a string
4. Convert a string to title case without using `.title()`
5. Remove all whitespace from a string
6. Count frequency of each character
7. Remove duplicate characters
8. Find first non-repeating character
9. Replace a word in a string with another
10. Check if two strings are anagrams

## Lists
11. Find the maximum and minimum in a list
12. Find the second largest number
13. Remove duplicates from a list
14. Reverse a list without using `.reverse()`
15. Sum all elements of a list
16. Sort a list without using `sorted()`
17. Merge two lists
18. Remove all odd numbers from a list
19. Find common elements between two lists
20. Count the occurrences of each element in a list

## Numbers
21. Check if a number is even or odd
22. Check if a number is prime
23. Find the factorial of a number (iterative)
24. Find the sum of digits of a number
25. Generate the first N Fibonacci numbers
26. Check if a number is an Armstrong number
27. Reverse the digits of an integer
28. Find the GCD and LCM of two numbers
29. Convert decimal to binary
30. Find all prime numbers in a range
